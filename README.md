# Staphylococcus-pseudintermedius-Genomics-Uruguay-Samples-2026
WGS (Illumina) was performed on 10 canine S. pseudintermedius strains. Assembly via Unicycler and annotation by Prokka. Phylogeny included >3500 NCBI genomes using Panaroo and FastTree. AMR genes, SCCmec, and MLST were identified using ResFinder, SCCmecFinder, and MLST tools.
